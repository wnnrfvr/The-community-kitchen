# Contributing to Community Cookbook

First off, thank you for being here! We want to make this project easy to work on. Please follow these steps to ensure your changes get merged smoothly.

## üõ† The Workflow

1. **Fork the Repo:** Click the 'Fork' button at the top right of this page.
2. **Clone your Fork:** `git clone https://github.com/YOUR-USERNAME/community-cookbook.git`
3. **Create a Branch:** Give it a descriptive name.
   `git checkout -b feature/add-taco-recipe`
4. **Make Changes:** Add your code or documentation.
5. **Commit & Push:**
   `git add .`
   `git commit -m "Added a delicious taco recipe to recipes.json"`
   `git push origin feature/add-taco-recipe`
6. **Open a Pull Request (PR):** Go to the original repository and click "New Pull Request".

## üìù Rules for Documentation
* All recipes added to `/data/recipes.json` must follow the existing JSON format.
* Images must be placed in `/assets` and be under 500kb.

## üêû Reporting Issues
If you find a bug, please open an **Issue** on GitHub. Describe the bug and provide steps to reproduce it.
